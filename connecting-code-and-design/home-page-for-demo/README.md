# Home Page For Demo

This folder isolates the pixel-focused homepage work into two pieces:

- `apps/homepage-demo`: a small browser app that renders just the homepage
- `apps/dark-mode-demo`: the inverted dark-mode variant of the homepage demo
- `packages/homepage`: the publishable npm package that owns the homepage implementation

From the workspace root, run `npm run dev:home-page-demo` to launch the focused demo app.
Run `npm run dev:dark-mode-demo` to launch the inverted dark-mode version.
